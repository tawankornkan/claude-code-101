# User Interview — ปอ
อายุ: 38 ปี | อาชีพ: Personal Trainer | ที่อยู่: กรุงเทพฯ
วันที่: 4 มิถุนายน 2569
บทบาท: Coach

---

**Q1: ตอนนี้คุณจัดการ client ยังไง? ใช้ tool อะไร track progress?**

มี client 15 คน ใช้ Google Sheets เป็นหลัก กรอกเองทุกอย่างหลัง session แล้วก็ Line Group ต่อ squad ปัญหาคือ client บางคนไม่ log เอง ต้องเตือนทุกสัปดาห์ เสียเวลามาก

**Q2: ปัญหาหลักในการดูแล squad คืออะไร?**

ปัญหาใหญ่สุดคือ data ไม่สะอาด บาง client บอกว่าทำแล้วแต่ไม่มีอะไร verify ทำให้ไม่รู้จริงๆ ว่าทำหรือเปล่า อีกอย่างคือตอน client เยอะขึ้น admin งานกิน 2–3 ชั่วโมงต่อสัปดาห์เลย

**Q3: ถ้ามีแอปที่ช่วย manage squad, log strength data, และมี leaderboard อัตโนมัติ คุณจะใช้ไหม?**

นี่คือสิ่งที่อยากได้มากเลย photo proof จะแก้ปัญหา verification ได้เลย leaderboard ก็ช่วย motivate client ได้โดยไม่ต้องพึ่งผมคนเดียว อยากได้ coach dashboard ที่เห็นภาพรวมทุก client ได้ทีเดียว ยินดีจ่าย 299–399 บาทต่อเดือนถ้า save เวลา admin ได้จริง
