# User Interview — ต้น
อายุ: 32 ปี | อาชีพ: Personal Trainer | ที่อยู่: กรุงเทพฯ
วันที่: 3 มิถุนายน 2569
บทบาท: Coach

---

**Q1: ตอนนี้คุณจัดการ client ยังไง? ใช้ tool อะไร track progress?**

ใช้ Line กลุ่มกับแต่ละ squad แล้วก็ spreadsheet ส่วนตัว client ส่งรูปมาใน Line แต่ดูยากมากเพราะ chat กับคุยเรื่องอื่นปนกัน บางทีหา progress ของ client คนไหนไม่เจอเลย

**Q2: ปัญหาหลักในการดูแล squad คืออะไร?**

ปัญหาหลักคือเวลา admin งาน ต้อง scroll Line หาข้อมูล, กรอก spreadsheet เอง, track ว่าใครออกครบหรือยัง ใช้เวลาไปกับ admin งานมากกว่า coach งานจริงๆ อยากได้ dashboard ที่เห็นภาพรวม squad ได้ทีเดียว

**Q3: ถ้ามีแอปที่ช่วย manage squad, log strength data, และมี leaderboard อัตโนมัติ คุณจะใช้ไหม?**

ใช้แน่นอน ถ้าลด admin time ได้จริง เวลาที่ประหยัดได้คือ value ที่ชัดเจนมากสำหรับ trainer client ก็จะได้รับ attention จากผมมากขึ้นด้วย ยินดีจ่าย 299 บาทต่อเดือนถ้า manage ได้ 10+ client โดยไม่ต้อง manual ทุกอย่าง
