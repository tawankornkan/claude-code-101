# Communication Style: Email

## Purpose
สื่อสารอย่างเป็นทางการ มี context ครบ เหมาะกับเรื่องที่ต้องการ record หรือส่งถึงคนนอกทีม

## Format Rules
- **Length:** กระชับ มี paragraph ไม่เกิน 3 อัน
- **Tone:** สุภาพ ชัดเจน มี clear ask
- **Structure:** เปิดด้วยบริบท → เนื้อหา → สิ่งที่ต้องการ
- **Avoid:** เกริ่นยาว, ข้อมูลที่ผู้รับไม่ต้องการ
