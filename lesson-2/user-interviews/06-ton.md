**Q2: เคยเจอ red flag จากคนที่เจอผ่าน app ไหม?**

รู้เรื่อง red flag ดีอยู่แล้ว แต่ก็ยอมรับว่าเคย ignore red flag เพราะคนนั้นหล่อมาก หัวเราะตอนพูด

**Q3: ถ้ามีแอปที่ช่วยวิเคราะห์ red flag คุณจะใช้ไหม และยินดีจ่ายไหม?**

กังวลเรื่อง AI bias ว่าจะ fair กับทุกคนไหม เสนอว่าควรให้ user define red flag เองได้เพราะแต่ละคน dealbreaker ไม่เหมือนกัน ยินดีจ่ายถ้า logic โปร่งใสและ customize ได้
